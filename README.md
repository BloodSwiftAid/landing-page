# SwiftAid Landing Page

SwiftAid is a premium, scalable coordination and intelligence platform designed to bridge the gap in blood access within Nigeria's healthcare system. It connects hospitals to verified blood sources in under 20 minutes, addressing the critical 1-in-4 maternal mortality rate caused by heavy bleeding.

## 🎨 Branding & Aesthetics
- **Primary Color (Teal)**: `#0f5f6e` (Professionalism, Medical Trust)
- **Accent Color (Gold)**: `#d59e4b` (Urgency, Quality, Premium)
- **Typography**: Inter (Body) & Outfit (Headings)

## 🚀 Quality & Performance
- **Modern UI**: Implemented with Glassmorphism and smooth animations.
- **Responsive**: Fully optimized for mobile, tablet, and desktop views.
- **High Impact**: Strategically designed sections to drive hospital onboarding and partner recruitment.

## 🛠 Tech Stack
- **Framework**: [React](https://reactjs.org/)
- **Bundler**: [Vite](https://vitejs.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🏃 How to Run Locally

1. **Clone the repository** (or navigate to the project folder).
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Start the development server**:
   ```bash
   npm run dev
   ```
4. **Open in browser**: Navigate to the URL shown in the terminal (usually `http://localhost:5173`).

## 📦 Project Structure
- `src/App.jsx`: Main landing page implementation with all core sections.
- `src/index.css`: Design system tokens, brand colors, and global styles.
